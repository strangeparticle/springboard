Springboard uses third-party open-source software.

Scope
-----
This notice file covers third-party application libraries redistributed with Springboard.
Springboard's own project license is provided separately in LICENSE.txt.

Included third-party application library families
-------------------------------------------------
- Kotlin Standard Library and Kotlin Test (org.jetbrains.kotlin:*)
- Compose Multiplatform / Compose Desktop (org.jetbrains.compose:*)
- AndroidX Lifecycle and related AndroidX libraries (org.jetbrains.androidx:*, androidx.*)
- Kotlin Coroutines (org.jetbrains.kotlinx:kotlinx-coroutines-*)
- Kotlin Serialization (org.jetbrains.kotlinx:kotlinx-serialization-*)
- Kotlinx Datetime (org.jetbrains.kotlinx:kotlinx-datetime)
- AtomicFU (org.jetbrains.kotlinx:atomicfu)
- Skiko (org.jetbrains.skiko:*)

--------------------
Desktop distributions include an embedded Java runtime image.
Its license and notice files are bundled automatically by the packaging toolchain
under the runtime legal directory inside the application bundle.
